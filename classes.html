A class in Java is a logical template used by the computer to create objects and to define object data types and methods. All objects in a given class will have the same methods or properties. A class in Java is the foundation that all other objects are based off of. The class groups sets of objects which share common characteristics/ behavior and common properties/attributes. Classes are the categories and the objects are the items within each category.
