What are classes used for?

Classes are used because it provides templates for creating objects, which can bind code to data. Classes have definitions of methods and data. It supports the inheritance property of Object Oriented Programming. The inheritance property allows code from one class to be part of another class without having to retype all of the code.

To relate this to real life, there can be a dog class. In this class, the object is the dog. The dog has attributes such as color, eye color, height, length, and weight. It also has commands that it knows such as sit, lay down, shake, come, and breathe. These commands are the methods.
